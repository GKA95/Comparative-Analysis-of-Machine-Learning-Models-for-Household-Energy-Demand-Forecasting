# Dataset Description

## Dataset

Individual Household Electric Power Consumption Dataset

Source:
UCI Machine Learning Repository

## Target Variable

Global Active Power

## Features

* Global Active Power
* Global Reactive Power
* Voltage
* Global Intensity
* Sub Metering 1
* Sub Metering 2
* Sub Metering 3

## Time Span

The dataset contains household electricity consumption measurements collected over several years and is widely used for energy forecasting research.

## Purpose

The dataset was used to develop and evaluate machine learning models for short-term household energy demand forecasting.
